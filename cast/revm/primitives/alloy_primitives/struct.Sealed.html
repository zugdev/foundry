Struct Sealed

pub struct Sealed<T> {
    inner: T,
    seal: FixedBytes<32>,
}
Expand description

A consensus hashable item, with its memoized hash.

We do not implement any specific hashing algorithm here. Instead types implement the Sealable trait to provide define their own hash.
