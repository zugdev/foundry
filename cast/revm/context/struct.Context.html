Struct Context

pub struct Context<BLOCK = BlockEnv, TX = TxEnv, CFG = CfgEnv, DB = EmptyDBTyped<Infallible>, JOURNAL = Journal<DB>, CHAIN = (), LOCAL = LocalContext>
where
    DB: Database,
    JOURNAL: JournalTr<Database = DB>,
    LOCAL: LocalContextTr,
{
    pub block: BLOCK,
    pub tx: TX,
    pub cfg: CFG,
    pub journaled_state: JOURNAL,
    pub chain: CHAIN,
    pub local: LOCAL,
    pub error: Result<(), ContextError<<DB as Database>::Error>>,
}
Expand description

EVM context contains data that EVM needs for execution.

Fields§

§block: BLOCK

Block information.

§tx: TX

Transaction information.

§cfg: CFG

Configurations.

§journaled_state: JOURNAL

EVM State with journaling support and database.

§chain: CHAIN

Inner context.

§local: LOCAL

Local context that is filled by execution.

§error: Result<(), ContextError<<DB as Database>::Error>>

Error that happened during execution.
