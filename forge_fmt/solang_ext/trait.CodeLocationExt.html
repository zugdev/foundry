Trait CodeLocationExt

Source 
pub trait CodeLocationExt {
    // Required method
    fn loc(&self) -> Loc;
}
Expand description

Returns the code location.

Patched version of [pt::CodeLocation]: includes the block of a [pt::FunctionDefinition] in its loc.
