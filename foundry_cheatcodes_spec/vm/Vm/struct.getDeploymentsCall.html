Expand description

Returns all deployments for the given contract on chainId

Sorted in descending order of deployment time i.e descending order of BroadcastTxSummary.blockNumber.

The most recent deployment is the first element, and the oldest is the last. Function with signature getDeployments(string,uint64) and selector 0x74e133dd.

function getDeployments(string calldata contractName, uint64 chainId) external view returns (address[] memory deployedAddresses);
