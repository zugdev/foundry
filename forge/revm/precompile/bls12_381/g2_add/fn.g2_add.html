    input: &[u8],

Expand description

G2 addition call expects 512 bytes as an input that is interpreted as byte concatenation of two G2 points (256 bytes each).

Output is an encoding of addition operation result - single G2 point (256 bytes). See also https://eips.ethereum.org/EIPS/eip-2537#abi-for-g2-addition
